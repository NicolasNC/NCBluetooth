# NCBluetooth
NCBluetooth是CoreBluetooth的一个学习demo，实现比较繁杂，能学习蓝牙的一个完整的通讯流程。
最近学习蓝牙，所以找了一些资料学习研究了一下蓝牙通讯的一个流程。写了一个小demo，demo效果见下图：
